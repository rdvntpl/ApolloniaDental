# ApolloniaDental
This is a Dental Health Center Project </br>
The Frontend built with EJS and Bootstrap </br>
ExpressJS, NodeJs and MongoDB on Docker

### Prerequisites
NodeJS, NPM, MongoDB are required to be installed.

### Setting Up
Clone the repo
   ```sh
   git clone https://github.com/rdvntpl/ApolloniaDental.git
   ```

### Running with Docker
For Docker users,</br>
Open terminal in project root folder </br>
Type this command
   ```sh
   docker compose up
   ```
Open Browser localhost:3000
